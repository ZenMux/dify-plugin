# ZenMux Dify Plugin Privacy Policy

This privacy policy explains how the ZenMux Dify Plugin collects, uses, and processes user data. Please read this policy carefully before using this plugin.

## Data Collection

This plugin collects and processes the folloing types of data:

### Configuration Information
- ZenMux Api Key Configuration
  - Purpose: Used for authentication with the zenmux.ai to access models.
  - Storage Location: Stored only in your Dify instance and will not be sent to any third-party services.

- Custom Model Configuration
  - Purpose: Used to access custom models provided by zenmux.ai.
  - Storage Location: Stored only in your Dify instance and will not be sent to any third-party services.

## Data Usage

The data collected by this plugin is used solely for the following purposes:
1. Enabling the connection between zenmux.ai and the Dify AI application.
2. Processing user messages and returning AI responses.

### Third-Party Services
This plugin needs to interact with the following third-party services:

- zenmux.ai
  - Shared Data: Api key, User messages, AI response content
  - Purpose: To access models provided by zenmux.ai
  - Privacy Policy: [ZenMux Privacy Policy](https://docs.zenmux.ai/privacy.html)

This plugin will not share your data with any third parties other than those listed above.

## Data Security

This plugin takes the following measures to protect user data security:
1. The plugin itself does not store the personal identity information.
2. Only temporarily saves and processes data for the necessary period.

## User Rights

As a user of the plugin, you have the right to:
1. Understand how the plugin processes user data.
2. Deactivate the plugin and delete related configurations at any time.

## Privacy Policy Updates

We may update this privacy policy from time to time. In the event of significant changes, we will notify users by updating the version number.

## Contact Information

If you have any questions about this privacy policy, please contact the plugin author.
